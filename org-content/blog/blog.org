This is a stub entry to demonstrate how =ox-hugo= metadata maps onto Astro.

- Change the =EXPORT_DATE= to the publish date you want to appear.
- Remove =EXPORT_HUGO_DRAFT: true= to publish the post.
- To add an image set =#+HUGO_CUSTOM_FRONT_MATTER+: heroImage "../public/..."= or update after export.

Use =C-c C-e H H= (or =org-hugo-export-to-md=) to export the subtree.
